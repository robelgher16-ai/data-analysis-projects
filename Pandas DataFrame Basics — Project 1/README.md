# Pandas DataFrame Basics — Project 1
A beginner-friendly Python project demonstrating essential **pandas** operations using a simple student dataset.

## 📋 Overview
This notebook walks through 17 common pandas tasks on a manually created DataFrame containing student names, marks, and gender. It is ideal for anyone learning data manipulation with pandas from scratch.

## 🗂️ Dataset
The dataset is created directly in the notebook as a Python dictionary and converted to a DataFrame.

| Column | Type | Description |
|--------|------|-------------|
| `Name` | string | Student name |
| `Marks` | integer | Marks out of 100 |
| `Gender` | string | Male / Female |

## 📌 Topics Covered

| # | Task |
|---|------|
| 1 | Display top 3 rows (`head`) |
| 2 | Display last 3 rows (`tail`) |
| 3 | Find shape of the dataset |
| 4 | Get dataset info (`info`) |
| 5 | Check for null values |
| 6 | Overall statistics (`describe`) |
| 7 | Find unique values in a column |
| 8 | Count unique values (`nunique`) |
| 9 | Value counts per category |
| 10 | Filter rows by condition (`between`) |
| 11 | Calculate average marks (`mean`) |
| 12 | Apply a custom function (`apply` + `lambda`) |
| 13 | Map values to new labels (`map`) |
| 14 | Drop a column |
| 15 | Print column names |
| 16 | Sort DataFrame by column |
| 17 | Filter rows and select specific columns |
